# 六角ナット（ナット高さ）の基礎知識

六角ナットはボルトと組み合わせて締結に使われる部品です。締結強度や用途に応じて形状・寸法が規格化されており、JIS B 1181 で定義されています。

## ナット高さ（m）とは

ナット高さ（m）は、ナットの上下面間の距離（厚み）を指します。高さが大きいほど締結強度は高くなりますが、設置スペースも必要になります。

## 1種・2種・3種の違い

JIS B 1181 では六角ナットを **1種・2種・3種** に分類しています。

- **1種**：標準的な高さ。一般用途に広く使われます。
- **2種**：1種よりやや高く、より高い締結力が必要な用途に使われます。
- **3種**：薄型（低形）。スペースが限られる箇所やロックナットとして使用されます。

## 呼び径とサイズの関係

呼び径（M○○）が大きくなるにつれ、ナット高さ・二面幅・対角寸法もそれぞれ大きくなります。各寸法は規格表で呼び径ごとに定められています。

## 本アプリの前提

本アプリのナット計算機は **JIS B 1181 の 1種** を前提とした寸法を参照しています。2種・3種の寸法については別途規格表をご確認ください。

## 注意事項

- JIS 規格は改正されることがあります。最新の規格値は JIS 原典または対応機関にてご確認ください。
- 製品によって公差の範囲内で寸法が異なる場合があります。実使用では現物確認を推奨します。
