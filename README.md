# calcnavi

計算ツールをまとめた Next.js 製の Web アプリです。

## 機能
- 複数の計算ページ（随時追加）
- ブラウザ内で完結（必要に応じて LocalStorage を利用）

## 技術スタック
- Next.js（App Router）
- TypeScript
- CSS（プロジェクト方針に準拠）
- デプロイ：Vercel（推奨）

## 開発

### 前提
- Node.js 18 以上
- npm

### セットアップ
```bash
git clone https://github.com/takumi-nakamura-eng/calcnavi.git
cd calcnavi
npm install
npm run dev
