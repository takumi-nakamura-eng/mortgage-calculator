<h2><span aria-hidden="true">💡</span><span>この記事でわかること</span></h2>
<ul>
  <li>並目ねじと細目ねじの違い</li>
  <li>ピッチ表記を図面上でどう読むか</li>
  <li>ボルト長さ計算の <code>3p</code> にどうつながるか</li>
</ul>

<h2><span aria-hidden="true">✅</span><span>結論</span></h2>
<ul>
  <li>ねじの呼び径に「×ピッチ」の表記がなければ並目（[JIS B 0205](https://kikakurui.com/b0/B0205-4-2001-01.html)）です。</li>
  <li>並目は流通量が多く入手性に優れるため、特別な理由がない限り並目を選ぶのが実務の基本です。</li>
  <li>細目を採用するのは、振動条件、微調整、薄肉部品など明確な理由がある場合に限ります。</li>
  <li>ボルト長さ計算では <code>先端余長 = 3p</code> の <code>p</code> に採用ねじのピッチを使います。</li>
</ul>

使い分けの考え方は [JIS B 0205](https://kikakurui.com/b0/B0205-4-2001-01.html) の規格値と、実務上の選び分けを整理した [MISUMI 技術情報](https://jp.misumi-ec.com/tech-info/categories/technical_data/td01/c1678.html) を基にしています。

<h2><span aria-hidden="true">📊</span><span>比較表</span></h2>

| 項目 | 並目 | 細目 |
|---|---|---|
| ピッチ | 大きめ（JIS B 0205 標準値） | 小さめ（JIS B 0207 規定値） |
| 入手性 | 高い（市販品の大半が並目） | 低め（受注生産や在庫限定の場合あり） |
| 主な用途 | 汎用締結・一般機械 | 振動対策・微調整・薄肉部品 |
| ゆるみやすさ | リード角が大きくゆるみやすい | リード角が小さくゆるみにくい |
| 加工難度 | 容易（汎用タップで対応） | やや高い（細目専用タップが必要） |

条件: M10 並目 `p = 1.5 mm`。

式: 先端余長 `= 3p`

計算: `3 × 1.5 = 4.5 mm`

結果: 先端余長は **4.5 mm**。この値をボルト長さ計算の式 `L = t + Hpw + Hsw + Hnut + 3p` に加算します。

<h2><span aria-hidden="true">⚠️</span><span>よくある見落とし</span></h2>

- M10 の並目ピッチ `1.5 mm` と M12 の並目ピッチ `1.75 mm` を取り違えて、先端余長が 0.75 mm ずれる
- 図面に「M10」とだけ記載されているとき、並目と細目の区別をせずに細目ピッチで計算してしまう
- 細目ナットに並目ボルトを組み合わせて締結不良を起こす（ピッチが異なるとねじ山が噛み合わない）
- 社内在庫の並目ボルトが欠品のとき、「ピッチが近い」という理由で細目を代用して強度不足になる

<h2><span aria-hidden="true">🧭</span><span>判断基準</span></h2>

`契約図書・組立図の指定 > JIS B 0205 並目ピッチ表 > メーカー推奨値 > 社内基準（競合時は保守側）`

<h2><span aria-hidden="true">❓</span><span>FAQ</span></h2>

<h3><span aria-hidden="true">💬</span><span>Q1. M表記だけなら並目ですか？</span></h3>
<p>A. 特記がない場合は並目扱いになるのが一般的です。図面に「M10×1.25」のようにピッチが明記されている場合のみ細目です。</p>

<h3><span aria-hidden="true">💬</span><span>Q2. 並目と細目を混在できますか？</span></h3>
<p>A. できません。呼び径が同じでもピッチが異なれば締結不良になります。並目ボルトには並目ナット、細目ボルトには細目ナットを必ず組み合わせてください。</p>
